# ANIMA

A browser-based music visualizer that turns audio into a living, reactive digital organism.

## Features

- 🎵 Drag & drop audio files
- 🔴 Live Spotify, YouTube, system, or microphone audio
- 🌌 Real-time audio-reactive visuals
- ⚡ Beat detection, particles, lightning, and waveform effects
- 📊 Built-in BPM, spectrum, oscilloscope, and spectrogram analysis
- 🎨 Multiple color palettes
- 🔒 100% local — no uploads, tracking, or accounts

## Controls

- `Space` — Play / Pause
- `F` — Fullscreen
- `I` — Toggle analytics
- `C` — Change colors
- `H` — Hide UI
- `← →` — Seek

Open the HTML file in a modern browser and start visualizing your music.
